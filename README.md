# 焼肉屋さん検索LINE BOT

位置情報をLINEで送ることで近くにある焼肉屋さんを教えてくれるLINE BOTです。

## 技術

- Golang
- LINE API
- ホットペッパー API

## 環境変数

使用するのに必要な環境変数は下記です。

- LINE_ACCESS_KEY：LINEのアクセスキー
- LINE_SECRET_KEY：LINEのシークレットキー
- HOTPEPPER_API_KEY：ホットペッパーAPIキー（https://webservice.recruit.co.jp/doc/hotpepper/reference.html）
