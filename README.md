# 焼肉屋さん検索LINE BOT(Yakiniku Sagasitai)

位置情報を送れば近くの焼肉屋さんを教えてくれるLINE BOTです。サクッと近くの焼肉屋さんが知ることができます。

下記のQRコードから追加することができます。

## 使用技術

- Golang
- LINE API
- ホットペッパー API
